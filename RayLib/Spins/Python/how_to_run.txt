How to run?

1. Make sure python3 is installed
2. If you are on linux/BSD you can run run_me.sh (Might also work for Macs tho I am not 100% sure how they work)

Additional note:
If somehow the script doesn't work or isn't provided for whatever reason you have to use python3 of venv manually like so:

UNIX:
python3 -m venv .venv
source ./.venv/bin/activate
pip install raylib
deactivate
./.venv/bin/python3 main.py
